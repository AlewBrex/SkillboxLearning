Временная сложность

Поиск минимального значения в массиве чисел длиной n, который отсортирован по возрастанию. = О(1)

Расчёт среднего значения в массиве чисел длиной n. = O(n)

Получение длины массива размером n. = O(1)

Задан список из n объектов, каждый из которых представляет собой банковский счёт - ArrayList<Bill>.
И есть класс и метод, с помощью которых можно получить общую сумму транзакций между первым и вторым счётом —
TransactionsCalculator.calculateTotalSum(Bill sourceBill, Bill destinationBill).
Этим методом алгоритм должен посчитать общую сумму переводов между всеми счетами из списка.
Какова будет временная сложность такого алгоритма? = O(n*n - 2)
